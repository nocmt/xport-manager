# 端口管理器 (Port Manager)

Visual Studio Code 的端口管理插件，支持查看端口、进程名称、PID，并可停止进程。

## 功能

- **查看运行端口**: 列出当前系统所有 TCP/UDP 监听端口。
- **显示进程信息**: 显示占用端口的进程名称和 PID。
- **停止进程**: 支持右键点击端口条目停止对应进程。
- **搜索过滤**: 支持按端口号、进程名或 PID 搜索。

## 支持系统

- Windows
- macOS
- Linux

## 使用方法

1. 点击活动栏上的 "端口管理" 图标。
2. 在视图中查看端口列表。
3. 点击标题栏的搜索图标进行过滤。
4. 右键点击条目选择 "停止进程"。

## 依赖

- Windows: 使用 `netstat` 和 `tasklist`。
- macOS/Linux: 使用 `lsof`。请确保系统已安装 `lsof`。
